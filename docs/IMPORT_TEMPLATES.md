# 数据导入CSV模板说明

## 模板文件位置
模板文件位于 `templates/` 目录下：
- `import_customers.csv` - 客户导入模板
- `import_contacts.csv` - 联系人导入模板
- `import_suppliers.csv` - 供应商导入模板
- `import_opportunities.csv` - 商机导入模板
- `import_contracts.csv` - 合同导入模板
- `import_invoices.csv` - 发票导入模板
- `import_payments.csv` - 回款导入模板
- `import_payables.csv` - 付款导入模板

> 注意：CSV文件可用Excel打开后另存为xlsx格式

## 模板工作表说明

### 1. 客户 (Customers)
| 字段 | 必填 | 说明 |
|------|------|------|
| 客户名称 | 是 | 客户公司名称 |
| 行业 | 否 | 如：制造业、服务业 |
| 来源 | 否 | 如：线上推广、朋友推荐 |
| 等级 | 否 | 如：A、B、C |
| 电话 | 否 | 联系电话 |
| 邮箱 | 否 | 电子邮箱 |
| 地址 | 否 | 公司地址 |
| 描述 | 否 | 备注信息 |

### 2. 联系人 (Contacts)
| 字段 | 必填 | 说明 |
|------|------|------|
| 联系人姓名 | 是 | 联系人姓名 |
| 所属客户 | 是 | 客户名称（必须已存在于系统中） |
| 职位 | 否 | 如：总经理、采购经理 |
| 电话 | 否 | 联系电话 |
| 邮箱 | 否 | 电子邮箱 |
| 是否主联系人 | 否 | TRUE/FALSE，默认FALSE |

### 3. 供应商 (Suppliers)
| 字段 | 必填 | 说明 |
|------|------|------|
| 供应商名称 | 是 | 供应商公司名称 |
| 联系人 | 否 | 供应商联系人姓名 |
| 电话 | 否 | 联系电话 |
| 邮箱 | 否 | 电子邮箱 |
| 地址 | 否 | 公司地址 |
| 描述 | 否 | 备注信息 |

### 4. 商机 (Opportunities)
| 字段 | 必填 | 说明 |
|------|------|------|
| 商机名称 | 是 | 商机名称 |
| 所属客户 | 是 | 客户名称 |
| 金额 | 否 | 商机金额，默认0 |
| 阶段 | 是 | PROSPECTING/QUALIFICATION/PROPOSAL/NEGOTIATION/CLOSED_WON/CLOSED_LOST |
| 概率 | 否 | 1-100，默认10 |
| 预计成交日期 | 否 | 格式：YYYY-MM-DD |
| 描述 | 否 | 备注信息 |

### 5. 合同 (Contracts)
| 字段 | 必填 | 说明 |
|------|------|------|
| 合同名称 | 是 | 合同名称 |
| 合同编号 | 否 | 留空则自动生成 |
| 合同类型 | 是 | SALES（销售合同）/ OUTSOURCING（委外合同） |
| 客户 | 销售合同必填 | 客户名称 |
| 供应商 | 委外合同必填 | 供应商名称 |
| 金额 | 否 | 合同金额，默认0 |
| 开始日期 | 否 | 格式：YYYY-MM-DD |
| 结束日期 | 否 | 格式：YYYY-MM-DD |
| 状态 | 否 | DRAFT/PENDING/SIGNED/ACTIVE/COMPLETED/CANCELLED，默认DRAFT |

### 6. 发票 (Invoices)
| 字段 | 必填 | 说明 |
|------|------|------|
| 发票号 | 是 | 发票编号（不可重复） |
| 关联合同 | 是 | 合同名称 |
| 发票类型 | 是 | SALES（开票/销售）/ PURCHASE（收票/委外） |
| 金额 | 是 | 不含税金额 |
| 税额 | 否 | 税额，默认0 |
| 价税合计 | 是 | 自动计算（金额+税额） |
| 开票日期 | 是 | 格式：YYYY-MM-DD |
| 到期日期 | 是 | 格式：YYYY-MM-DD |
| 状态 | 否 | DRAFT/ISSUED/PAID/OVERDUE/CANCELLED，默认DRAFT |

### 7. 回款 (Payments) - 仅销售合同
| 字段 | 必填 | 说明 |
|------|------|------|
| 关联合同 | 是 | 销售合同名称 |
| 关联发票 | 否 | 发票号（可选） |
| 金额 | 是 | 回款金额 |
| 回款日期 | 是 | 格式：YYYY-MM-DD |
| 回款方式 | 否 | 如：银行转账、现金 |
| 备注 | 否 | 备注信息 |
| 状态 | 否 | PENDING/COMPLETED/FAILED，默认PENDING |

### 8. 付款 (Payables) - 仅委外合同
| 字段 | 必填 | 说明 |
|------|------|------|
| 关联合同 | 是 | 委外合同名称 |
| 金额 | 是 | 应付金额 |
| 到期日期 | 是 | 格式：YYYY-MM-DD |
| 付款日期 | 否 | 格式：YYYY-MM-DD |
| 付款方式 | 否 | 如：银行转账 |
| 备注 | 否 | 备注信息 |
| 状态 | 否 | PENDING/COMPLETED/FAILED，默认PENDING |

## 导入注意事项
1. 所有日期格式请使用 YYYY-MM-DD（如：2024-01-01）
2. 必填字段必须填写，否则会导致导入失败
3. 关联字段（如客户名称、合同名称）必须已存在于系统中
4. 客户名称、供应商名称、合同名称等如重复，系统会跳过或覆盖
5. 导入前请备份现有数据
